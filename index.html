Questions fréquentes

Combien de temps pour installer l'agent ? +

Entre 3 et 5 jours après validation de vos infos. Je vous envoie un formulaire simple à remplir, puis je configure et teste l'agent avant mise en ligne.

Est-ce qu'il y a un engagement ? +

Non. L'abonnement est sans engagement, vous pouvez résilier avec 1 mois de préavis. L'installation est facturée une seule fois.

Et si l'agent répond mal à une question ? +

Vous pouvez me signaler toute réponse inappropriée, je corrige dans les 24h. L'agent apprend avec les corrections. Pour les sujets sensibles, il redirige automatiquement vers vous.

Où sont hébergées les données ? +

L'agent tourne sur Vercel (hébergeur mondial) avec API Claude (Anthropic). Aucune donnée client n'est conservée au-delà de la conversation, sauf si vous demandez un historique.

Je n'ai pas de site web, est-ce que ça marche ? +

Oui. L'agent peut être accessible via un lien partageable à mettre sur votre profil Google, Instagram, ou Facebook. Il peut aussi être branché directement sur WhatsApp Business.
